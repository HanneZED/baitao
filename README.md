# 肥窝 | 水汽白桃 power by 温尔文雅

这是“水汽白桃”的个人小窝网站，用来展示橱窗作品、视频/图文攻略、图片小盒、图片墙、联系方式，以及一个可拖动互动的小桃宠物。

这份说明尽量按“完全不懂 GitHub、前端开发也能照着做”的方式写。

## 最常用的更新方式

以后更新首页内容，优先用网页里的“更新内容”按钮，不需要手动改代码。

1. 打开网站首页。
2. 找到“应用 & 视频”区域。
3. 点击“更新内容”。
4. 按表单填写内容。
5. 输入管理密钥。
6. 点击“保存到网站”。
7. 等待 Vercel 自动重新部署，通常几十秒到几分钟后线上页面会更新。

## 第一次使用前要配置什么

网页保存功能需要 Vercel 帮你把内容写回 GitHub，所以要先在 Vercel 项目里配置环境变量。

### 必填环境变量

在 Vercel 项目后台进入 `Settings` -> `Environment Variables`，添加：

- `SITE_ADMIN_KEY`：你自己设置的管理密钥。以后网页里点“保存到网站”时就输入这个。
- `GITHUB_TOKEN`：GitHub 的访问令牌，用来允许网站把新内容提交到仓库。

### 可选环境变量

如果仓库仍然是当前这个，可以不填：

- `GITHUB_REPO`：默认是 `HanneZED/baitao`
- `GITHUB_BRANCH`：默认是 `main`

### GitHub Token 怎么准备

1. 打开 GitHub。
2. 进入头像菜单里的 `Settings`。
3. 找到 `Developer settings`。
4. 进入 `Personal access tokens`。
5. 创建一个 Fine-grained token。
6. Repository 选择这个网站仓库。
7. 权限给 `Contents: Read and write`。
8. 生成后复制 token，填到 Vercel 的 `GITHUB_TOKEN`。

注意：token 只会放在 Vercel 后台，不要发给别人，也不要写进网页文件里。

## 如何添加视频链接

1. 点击首页“更新内容”。
2. 在“添加视频 / 图文攻略”里选择“视频链接”。
3. 粘贴 B 站、小红书或其他网页链接。
4. 点击“读取链接封面”。
5. 如果读取成功，会自动填标题、简介和封面。
6. 如果读取失败，可以手动填写标题、简介，并上传封面。
7. 点击“添加到首页”。
8. 确认页面预览没问题后，输入管理密钥并点击“保存到网站”。

说明：封面读取依赖对方网页是否提供 `og:image` 这类预览信息。有些平台会限制读取，失败时手动上传封面即可。

## 如何添加图文攻略

1. 点击首页“更新内容”。
2. 在“添加视频 / 图文攻略”里选择“图文攻略”。
3. 填标题、分类、封面、简介。
4. 在“正文”里写攻略内容。
5. 点击“添加到首页”。
6. 页面会先在本地预览这张新卡片。
7. 确认后输入管理密钥，点击“保存到网站”。

如果你填写了外部链接，卡片会打开原链接；如果写了正文，卡片会在本站弹出子页面，不会打断 BGM。

## 如何添加图片小盒

图片小盒适合做“超可爱表情包”“画稿与灵感”这种点开后展示一组图片的卡片。

1. 点击首页“更新内容”。
2. 在“添加可点开的图片小盒”里填标题和说明。
3. 上传封面，或者让第一张图片自动当封面。
4. 选择要放进子页面的多张图片。
5. 点击“添加小盒”。
6. 确认预览没问题后，输入管理密钥并保存。

保存后，访客点开这个小盒，会在本站打开子页面。BGM 和小桃宠物会继续保留并且可以操作。

## 如何更新图片墙

1. 首页找到“图片小盒”入口。
2. 点开后进入图片墙。
3. 点击“上传图片”添加新图。
4. 拖动相纸调整位置。
5. 需要删除时，点相纸右上角的红色按钮。
6. 点击“保存图片墙”。
7. 输入管理密钥并保存。

图片很大时建议先压缩，否则上传和保存会比较慢。

## 如何删除内容

1. 点击首页“更新内容”。
2. 在“当前内容”列表里找到要删的内容。
3. 点击“移除”。
4. 确认页面预览已经消失。
5. 输入管理密钥并点击“保存到网站”。

没有点击“保存到网站”之前，只是当前浏览器预览变了，线上网站不会真的变化。

## 常见问题

### 点保存后没有立刻更新

这是正常的。保存会先提交到 GitHub，再触发 Vercel 部署。等几十秒到几分钟后刷新页面即可。

### 保存失败

按顺序检查：

1. 管理密钥是不是和 Vercel 的 `SITE_ADMIN_KEY` 一样。
2. Vercel 里有没有配置 `GITHUB_TOKEN`。
3. GitHub Token 是否有仓库 `Contents: Read and write` 权限。
4. Token 是否过期。
5. 仓库和分支是不是 `HanneZED/baitao`、`main`。

### 读取链接封面失败

这通常不是网站坏了，而是对方平台不允许读取，或者没有提供网页预览图。手动上传封面即可。

### BGM 没有自动播放

浏览器通常不允许网页自动播放声音，所以需要用户点一次播放按钮。切换子页面时，BGM 不会因为页内弹层而中断。

### 小桃宠物挡住内容

小桃可以直接拖动到别的位置。长时间不理它时，会播放被冷落动画，并一直显示“呜呜~~”，直到再次点击或拖动它。

## 文件大概放在哪里

- `index.html`：首页结构。
- `styles.css`：所有视觉样式。
- `app.js`：网页交互、BGM、小桃宠物、内容管理逻辑。
- `data/site-content.json`：首页可维护内容、图片小盒、图片墙数据。
- `api/content.js`：保存内容到 GitHub 的接口。
- `api/metadata.js`：尝试读取外部链接标题、简介和封面的接口。
- `img/`：图片素材。
- `sound/BGM/`：背景音乐。

日常维护一般只需要使用网页里的“更新内容”，不用直接打开这些文件。
