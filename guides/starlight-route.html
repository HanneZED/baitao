游戏路线 / 收集记录

星光地图 收集路线

这套模板适合记录开放世界或养成游戏里的每日路线：先扫高收益点，再补材料、宝箱和隐藏任务， 最后做复盘，避免每天重新想一遍从哪里开始。

目录 路线准备 推荐顺序 检查清单 相关攻略

01 / Setup

先把地图拆成三个小圈。

每天只需要确认体力、传送点、当日刷新材料和限时任务。把地图拆成“主线圈”“材料圈”“顺手圈”， 这样路线不会越做越长。

02 / Route

推荐顺序

  1. 从离每日任务最近的传送点开始，先完成有时间限制的内容。
  2. 沿路收高价值材料，低价值点只在顺路时记录。
  3. 把隐藏宝箱、解谜和一次性任务放到周末清理。
  4. 路线结束后给今天的耗时、掉落和缺口做 3 行复盘。

03 / Checklist

每次出发前看一眼。

体力是否已经消耗或规划好用途
背包材料是否接近上限
今日限时任务是否已标记
截图点和收藏点是否已经备份
