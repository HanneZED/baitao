作品资料 标签系统

这页是给个人作品和攻略用的归档模板。核心目标是：以后想找某张图、某条路线、某个教程时， 不需要靠记忆硬翻。

返回攻略库
目录 标签层级 归档步骤 维护清单 相关攻略

01 / Tags

标签不要超过三层。

推荐使用“类型、主题、状态”三层标签。比如：作品 / 头像 / 已发布，攻略 / 游戏路线 / 待更新。 层级太多会变成新的负担。

02 / Flow

归档步骤

  1. 先按内容类型分入口：作品、攻略、素材、灵感、待办。
  2. 每个条目保留标题、标签、链接、状态、最后更新时间。
  3. 截图类资料统一写来源和用途，避免以后忘记为什么收藏。
  4. 每月只做一次清理，不把归档变成每天的压力。

03 / Maintain

维护时只问四个问题。

这个条目以后真的会再看吗
标签是否能让 3 个月后的自己找回来
是否有外部链接或源文件备份
状态是否需要改成完成、暂停或废弃
